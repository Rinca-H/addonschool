オリジナルのブロック


＋ もくじ
もくじ ・普通のブロックを追加する
・コンポーネント紹介
・イベントを使う
・パーミュテーションを使う

　オリジナルのブロックを追加する手順を説明していきます。

普通のブロックを追加する

　まずは「普通のブロック」を追加していきます。目立つ特徴のない、立方体の装飾ブロックのようなブロックです。もっとハデなものを作りたくてもはじめはこれを作ったほうがいいでしょう。先に基盤を作っておくことで、他のブロックを作るときに「普通のブロック」のファイルをコピーしてそこから発展させて楽に作ることができます。
　本当になにも特徴がないブロックなら以下のJSONで追加できます。 

{
  "format_version": "1.16.100",
  "minecraft:block": {
    "description": {
      "identifier": "ex:my_block"
    },
    "components": {}
  }
}

　しかしこれだとさすがに特徴が無さすぎです。サバイバルで壊せばそのブロック本体をドロップし、スライムブロックのように壊すのも一瞬です。
　そこでいくつか基本のコンポーネントは格納しておきましょう。 

{
  "format_version": "1.16.100",
  "minecraft:block": {
    "description": {
      "identifier": "ex:my_block"
    },
    "components": {
      "minecraft:destroy_time": 0.5,
      "minecraft:explosion_resistance": 10.0,
      "minecraft:map_color": "#64ec86"
    }
  }
}

minecraft:destroy_time

　そのブロックの破壊にかかる秒数を小数で指定します。ちなみに土を素手で掘ると0.75秒、黒曜石をダイヤのつるはしで掘ると9.4秒かかります。 　爆発耐性の強さによって若干長くなったりします。まだ計算式は見つかっていません。

minecraft:explosion_resistance

　そのブロックの爆発耐性を小数で指定します。

minecraft:map_color

　そのブロックの地図上での表示色をカラーコードで指定します。このコンポーネントを消すと、そもそも地図に表示されなくなります。
