JSONとは？


＋ もくじ
もくじ ・基礎中の基礎
・構文のルール
・このサイトでの指示

　JSONとは、アドオンを作るために使われているデータを構成するのに必要な構文のことです。アドオンの中でもっともデリケートな部分で、一文字のミスも許されません。たとえばオリジナルのモブを追加しようとしたときに「, (カンマ)」が一つないだけでそのモブが一切追加されなくなってしまうということはよくあることで、これは誇張ではなく本当です。 きちんとルールを身に付け、余計なミスのせいであきらめてしまうといったことのないようにしましょう！

基礎中の基礎

　JSONというものはこんな形をしています。 

{
  "format_version": "1.12",
  "minecraft:recipe_shaped": {
    "description": {
    "identifier": "minecraft:acacia_boat"
    },
    "tags": [ "crafting_table" ],
    "pattern": [
      "#P#",
      "###"
    ],
    "key": {
      "P": {
        "item": "minecraft:wooden_shovel"
      },
      "#": {
        "item": "minecraft:planks",
        "data": 4
      }
    },
    "result": {
      "item": "minecraft:boat",
      "data": 4
    }
  } 
}
